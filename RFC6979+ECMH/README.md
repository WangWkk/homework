## 项目代码说明
- SM2.py文件中包含SM2算法密钥生成函数，利用二倍点运算实现的椭圆曲线上的加法函数，其中生成公钥时k是根据RFC6979生成。
- ECMH.py文件调用SM2.py中椭圆曲线上的加法函数，实现ECMH方案。
