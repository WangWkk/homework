## 项目代码说明

- 2P_sign.py文件模拟了实际网络中2P签名过程。
- 需将2P_sign.py和SM2.py文件(位于“RFC6979+ECMH”文件夹中)放于同一文件夹下运行，签名时间较长，请耐心等待。
- 代码内容说明详见代码内注释。
